About

The Workshop on Compiler Oriented Software Security (CSS 2025) brings together researchers, practitioners, and tool developers to discuss the latest advancements in compiler design, automated bug detection, program analysis, and testing methodologies that will help in improving software security. The workshop will serve as a forum for exchanging ideas, presenting research, and fostering collaborations in the field of software security.

Call for Papers

We invite high-quality submissions on topics related to compiler based software security, including (but not limited to):

  • Automated bug synthesis and benchmarking
  • Symbolic execution and model checking
  • Fuzz testing and hybrid techniques
  • Testing for AI/ML-based systems
  • Compiler Optimizations
  • Code Generation

Submission Details

Maximum 2 pages (excluding references) in the standard ACM format. Submission Portal.
