About

The ACM Workshop on Automated Software Testing (AST 2025) brings together researchers, practitioners, and tool developers to discuss the latest advancements in automated bug detection, program analysis, and testing methodologies. The workshop will serve as a forum for exchanging ideas, presenting research, and fostering collaborations in the field of software testing.

Call for Papers

We invite high-quality submissions on topics related to automated software testing, including (but not limited to):

  • Automated bug synthesis and benchmarking
  • Symbolic execution and model checking
  • Fuzz testing and hybrid techniques
  • Testing for AI/ML-based systems
  • Applications in cybersecurity and blockchain

Papers should be submitted through the EasyChair system: Submit Here.
